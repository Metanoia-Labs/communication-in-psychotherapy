# Приложение C. Чеклисты, карты наблюдений, рабочие формы

Практические инструменты для структурирования работы и самоконтроля. См. также [Главу 23](./chapter23.md) с подробным описанием чеклистов и протоколов.

## Как пользоваться формами (чтобы не выпадать из контакта)

- **Эти формы — прежде всего для подготовки и разбора**, а не для того, чтобы “держать глазами” всю сессию.
- Во время сессии выберите **1–2 маркера** для намеренного наблюдения (например: контакт + темп).
- Простая формула: **«Одна сессия — один фокус процесса для наблюдения»** (см. [Главу 18](./chapter18.md)).
- Чеклист — **инструмент тренировки внимания**, а не экзамен: “ошибка” = данные, цель — увидеть повторяющиеся паттерны (см. [Главу 16](./chapter16.md)).

### Мини‑протокол (уровень входа)

**60 секунд до сессии:**
- моё состояние (0–10),
- контрперенос (1 слово),
- главная задача на сегодня (1 фраза).

**3 минуты после сессии:**
- самый живой момент,
- где упал контакт,
- что я сделал(а),
- что попробую иначе в следующий раз.

### Этическая заметка о записях

Если формы используются для супервизии/обучения, соблюдайте конфиденциальность: **обезличивайте**, фиксируйте минимум необходимых деталей, храните записи безопасно, соблюдайте правила организации/программы.

## C.0. “Один лист на сессию” (One-page session note)

Форма‑минимум, которую удобно распечатать и использовать как **одну страницу** на сессию: до → начало → фокус → 2–3 наблюдения → завершение. Это снижает перегруз и делает записи пригодными для супервизии.

### До сессии (60 секунд)

- **Моё состояние (0–10):** ________
- **Контрперенос (1 слово):** __________________
- **Риск (1 слово):** __________________

### Красные флаги безопасности (стоп‑сигналы)

Если отмечаете один пункт ниже, приоритет: **стабилизация / рамка / направление / супервизия**, а не углубление:
- [ ] признаки диссоциации / “не здесь”
- [ ] суицидальные высказывания / самоповреждения
- [ ] резкая эскалация агрессии / угроза
- [ ] клиент уходит “разобранным” в конце сессии

### Начало сессии (1 минута)

- **Тон:** тревога / грусть / злость / стыд / другое: ________
- **Контакт:** с собой / со мной / с процессом (что вижу): __________________
- **Темп:** ускорение / замедление / ровно: ________

### Фокус (1 фраза)

- **Главная задача на эту сессию:** ________________________________________

### Наблюдения (2–3 строки: факт → гипотеза → проверка → интервенция)

1) **Факт:** ____________________  **Гипотеза (1–2):** ____________________  
   **Проверка:** ____________________  **Интервенция:** ____________________

2) **Факт:** ____________________  **Гипотеза (1–2):** ____________________  
   **Проверка:** ____________________  **Интервенция:** ____________________

3) **Факт:** ____________________  **Гипотеза (1–2):** ____________________  
   **Проверка:** ____________________  **Интервенция:** ____________________

### Завершение (1–2 минуты)

- **Интеграция (1 фраза):** ________________________________________________
- **Следующий шаг (1 фраза):** ____________________________________________

## C.1. Чеклист состояния психолога перед сессией

### Моё состояние
- [ ] Я чувствую себя отдохнувшим и готовым к работе
- [ ] Уровень энергии достаточен для сессии
- [ ] Нет физического дискомфорта, который может отвлекать
- [ ] Я способен быть полностью присутствующим

### Моя тревога
- [ ] Я осознаю уровень своей тревоги
- [ ] Тревога находится в допустимых пределах
- [ ] Я могу работать с этой тревогой
- [ ] Тревога не будет мешать процессу

### Мои ожидания
- [ ] Я понимаю свои ожидания от сессии
- [ ] Я готов к тому, что процесс может пойти не так, как я ожидаю
- [ ] Я не фиксирован на определённом исходе

### Мой перенос/контрперенос
- [ ] Я осознаю свои реакции на этого клиента
- [ ] Я готов работать с собственными реакциями
- [ ] Я понимаю, какие эмоции вызывает у меня этот клиент

### Уровень присутствия
- [ ] Я могу быть полностью включённым в процесс
- [ ] Нет отвлекающих мыслей или переживаний
- [ ] Я готов быть "здесь и сейчас"

## C.2. Чеклист состояния клиента в начале сессии

### Эмоциональный тон
- Тревога / Спокойствие
- Грусть / Радость
- Раздражение / Дружелюбие
- Напряжение / Расслабленность
- Другое: ___________

### Невербалика
- Поза (открытая/закрытая, напряжённая/расслабленная)
- Взгляд (контакт/избегание)
- Жесты, микродвижения
- Дыхание (учащённое/ровное)

### Ритм речи
- Быстрый / Медленный
- Прерывистый / Плавный
- Громкий / Тихий

### Контактность
- [ ] Есть ли контакт со мной?
- [ ] Есть ли контакт клиента с самим собой?
- [ ] Есть ли контакт с процессом?

Подсказки‑примеры:
- **контакт с собой**: «ощущаю/чувствую/замечаю…», клиент различает переживания;
- **контакт со мной**: слышит реплики, реагирует, выдерживает паузу;
- **контакт с процессом**: держит линию разговора, можно договориться о фокусе.

### Уровень энергии
- Высокий / Низкий / Нормальный

### Признаки защиты
- [ ] Рационализация
- [ ] Избегание
- [ ] Юмор
- [ ] Интеллектуализация
- [ ] Другие защитные механизмы: ___________

### Микромаркеры раннего разрыва контакта (микросигналы)

- [ ] внезапная улыбка “не к месту”
- [ ] “правильные слова” без живости/отклика
- [ ] резкое ускорение/замедление темпа
- [ ] взгляд “мимо” / избегание в критический момент
- [ ] тело разворачивается/отодвигается/закрывается

## C.3. Чеклист фокуса сессии

- [ ] **Что является темой?** 
  - Основная тема сегодняшней сессии: ___________
  - Связь с общей темой терапии: ___________

- [ ] **Что является задачей?**
  - Что мы пытаемся достичь: ___________

- [ ] **Что является динамикой?**
  - Что происходит между клиентом и мной: ___________

- [ ] **Какой уровень глубины?**
  - Поверхностный / Эмоциональный / Структурный

- [ ] **Какая стратегия?**
  - Контейнирование / Прояснение / Интерпретация / Поддержка / Другое

Правило выбора “в одну фразу”:
- перегруз/высокий аффект → контейнирование,
- хаос/распад связности → структурирование,
- избегание → мягкое углубление,
- разрыв контакта → метакоммуникация/ремонт.

## C.5. Чеклист завершения сессии

- [ ] Снижена ли интенсивность эмоций?
- [ ] Есть ли интеграция материала?
- [ ] Сформулирован ли результат/вывод сессии?
- [ ] Клиент стабилен и может покинуть кабинет?
- [ ] Обозначена ли перспектива/следующая сессия?
- [ ] Рамка соблюдена (время, границы)?
- [ ] Клиент готов к завершению?

## C.7. Рабочие формы для супервизии

### Описание ситуации
- Контекст: ___________
- Что происходило: ___________
- Реплики клиента: ___________
- Реплики психолога: ___________

### Гипотезы
- Что я думаю происходило: ___________
- Какие процессы были активны: ___________

### Динамика
- Изменения контакта: ___________
- Разрывы: ___________
- Приближения/отдаления: ___________

### Интервенции
- Что я делал: ___________
- Какие интервенции использовал: ___________

### Самоанализ ошибок
- Что я сделал неправильно: ___________
- Где я потерял контакт: ___________
- Какие ошибки допустил: ___________

### План улучшений
- Что нужно изменить: ___________
- Над чем работать: ___________
- Какие навыки развивать: ___________

### Запрос и “стыдная зона”

- **Мой запрос к супервизору (1–2 вопроса):** _______________________________
- **Что я не решаюсь сказать/показать (стыдная зона):** ______________________
