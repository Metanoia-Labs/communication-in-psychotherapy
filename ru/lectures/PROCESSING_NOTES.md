# Заметки по обработке глав

Этот файл создан для отслеживания процесса обработки глав курса.

## Статус обработки

- ✅ chapter01.md - Готов
- ⏳ chapter02-chapter23.md - Требуют обработки

## Что нужно добавить в каждую главу:

1. Блок о вычитке в начале файла
2. Раздел "Проблемы материала лекции" (если отсутствует)
3. Дополнение литературы полными библиографическими ссылками
4. Исправление ссылок на другие главы (относительные пути)
5. Проверка наличия всех элементов:
   - Введение
   - Проблематика
   - Материал лекции
   - Проблемы материала лекции
   - Вопросы к обсуждению
   - Задания
   - Литература

## Формат блока о вычитке:

```markdown
> [!WARNING] Материал разработан с помощью ИИ
> Этот материал был разработан с использованием искусственного интеллекта и требует вычитки человеком.
> Статус вычитки: Не вычитано
```

## Формат ссылок:

Использовать относительные пути, например:
- `[Глава 1](./chapter01.md)`
- `[БЛОК I](./block01.md)`
- `[Оглавление](./contents.md)`
